# CYBERPUNK 2077 — Интерактивный фан-сайт

> Полностью статический фронтенд-проект в стиле киберпанк. Без фреймворков, без бэкенда — чистый HTML, CSS и JavaScript.

---

## Что сделано

### Страницы и контент

- **Главная страница** — герой-секция с анимацией цифрового дождя, интерактивные карточки персонажей, районов, банд и корпораций, встроенный музыкальный плеер с треками из игры
- **16 страниц персонажей** — V, Джонни Сильверхенд, Джуди, Панам, Ривер, Рог, Керри, Джеки, Такемура, Альт, Адам Смэшер, Виктор, Мисти, Т-Баг, Эвелин, Делемейн — у каждого своя цветовая схема, биография, статы и навыки
- **База данных банд** — Maelstrom, Valentinos, Tyger Claws, Animals, 6th Street и другие с фильтрацией и детальными карточками
- **База данных корпораций** — Arasaka, Militech, Kang Tao, Biotechnica, Petrochem, Trauma Team
- **Карта районов Night City** — интерактивная SVG-карта с 7 районами, подсветкой территорий банд и инфо-панелью
- **База оружия** — классификация по типам с характеристиками и фильтрами
- **Киберимпланты** — каталог с категориями и поиском
- **Риперы** — карта с пинами, список докторов с их специализациями

### Технические решения

#### Дизайн и анимации
- Кастомный курсор с эффектом прицела (точка + крестик + кольцо), следящий за мышью
- Canvas-анимация цифрового дождя в стиле матрицы на страницах персонажей
- CRT-эффект с вертикальными полосами сканирования и виньеткой
- Неоновые свечения, glitch-анимации и переходы через CSS `@keyframes`
- Фоновое фото Night City с parallax-эффектом через `position:fixed`
- Динамические цветовые схемы через CSS-переменные (`--ac`, `--cyan`, `--mag`) — каждый персонаж в своём цвете

#### Интерактивность
- Фильтрация карточек банд и корпораций без перезагрузки страницы (живой поиск + кнопки категорий)
- Slide-in детальная панель при клике на карточку (transform translateX)
- SVG-карта районов с hover-подсветкой, тултипами и кликабельными полигонами
- Музыкальный плеер: воспроизведение через YouTube iframe API, анимация прогресс-бара, список треков
- Анимация счётчиков цифр при появлении блока в viewport (IntersectionObserver)
- Анимация полосок статистики персонажей при скролле
- Reveal-анимации элементов при входе в viewport

#### Мобильная адаптация
- Hamburger-меню с полноэкранным overlay на всех 22 страницах
- Адаптивные гриды: 3 колонки → 2 → 1 в зависимости от ширины экрана
- Фильтр-панели на мобиле переключаются кнопкой с slide-in анимацией
- Кастомный курсор скрывается на тач-устройствах (`pointer:coarse`)
- Визуальная обратная связь при тапе (`:active` с opacity + scale)
- Предотвращение zoom на iOS через `font-size:16px` на инпутах
- Скрытие лишних элементов навигации на маленьких экранах

### Стек

| Технология | Использование |
|---|---|
| HTML5 | Разметка всех страниц |
| CSS3 | Анимации, grid, flexbox, CSS-переменные, медиазапросы |
| Vanilla JavaScript | Интерактивность, DOM-манипуляции, API |
| SVG | Интерактивная карта районов |
| Canvas API | Анимация цифрового дождя |
| YouTube iframe API | Встроенный музыкальный плеер |
| Google Fonts | Orbitron, Rajdhani |
